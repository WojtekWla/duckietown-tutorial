# Duckietown Tutorial

My work for the **ETH Zürich "Self-Driving Cars with Duckietown"** MOOC — a hands-on
course on autonomous systems, where you program a small robot (a Duckiebot) to
perceive, plan, and drive itself around the Duckietown environment.

This repo holds the exercises, experiments, and notes I write while going through
the course.

## Topics covered

- Robotics and autonomous systems fundamentals
- Computer vision and perception
- Localization, planning, and control
- Putting it together to drive a Duckiebot autonomously

## Setup

This project uses [uv](https://docs.astral.sh/uv/) and Python 3.11+.
